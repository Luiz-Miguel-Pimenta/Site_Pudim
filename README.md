# 🍮 Site do Pudim

Um site de conteúdo estático criado para apresentar a história, a receita e imagens do clássico pudim de leite condensado brasileiro, desenvolvido com HTML e CSS puro.

## 📋 Sobre o Projeto

O **Site do Pudim** é uma página informativa e visualmente agradável dedicada inteiramente ao pudim — um dos sobremesas mais queridas do Brasil. O site apresenta a origem histórica do prato, a receita completa passo a passo e uma galeria com fotos apetitosas. É um projeto front-end estático, sem uso de JavaScript, focado na estrutura e estilização do conteúdo.

## 📑 Seções do Site

### 📖 História
Apresenta a origem do pudim desde o Império Romano, passando pela influência portuguesa dos conventos, até a popularização no Brasil com o leite condensado a partir da década de 1930. Destaques incluem curiosidades como a origem da palavra "pudim" (do francês *boudin*) e a lenda do **Pudim Abade de Priscos**.

### 🧁 Receita
Contém a receita completa do pudim de leite condensado, dividida em:
- **Ingredientes** necessários
- **Modo de preparo** detalhado em 6 passos, incluindo preparo da calda de caramelo, mistura no liquidificador, coagem, montagem e cozimento em banho-maria a 180°C

### 🖼️ Galeria de Imagens
Exibe uma galeria com 6 fotos do pudim finalizado.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura e conteúdo da página
- **CSS3** — Estilização e layout
- **Google Fonts** — Fonte **Fredoka** para uma identidade visual lúdica e convidativa

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Luiz-Miguel-Pimenta/Site_Pudim.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Site_Pudim
   ```

3. Abra o arquivo `index.html` no seu navegador — não é necessário nenhum servidor ou instalação adicional.
